Kommentare

Kommentare können in JavaScript mit 

//
bzw. 
/* ... */
gebildet werden.

Variablen

Variablen werden mit dem Schlüsselwort 

var
eingeleitet. Im Gegensatz zu Java bezieht sich die Sichtbarkeit von Variablen nicht auf den aktuellen Block sondern auf die aktuelle Funktion. Werden Funktionen in einander verschachtelt so können die inneren Funktionen auf die Variablen der äußeren zugreifen.

Ausgaben auf der Konsole

Mit dem Befehl 

console.log
kann man sich Variableninhalte und Texte auf der JavaScript Konsole des Browsers ausgeben lassen.

Funktionen definieren

Mit dem Schlüsselwort 

function
kann man Funktionen in JavaScript definieren. In Klammern kann man Parameter angeben. Im Gegensatz zu Java werden keine Typen angegeben. Mit 
return
kann ein Rückgabewert zurückgegeben werden.

Arrays

Mit den eckigen Klammern können in JavaScript Array angelegt werden (wie in Java)

HashArrays

Mit Hilfe von HashArrays kann man Schlüssel-Wert Paare in JavaScript abbilden (technisch sind das auch Objekte in JavaScript). Solche HashArrays werden mit geschweiften Klammern gebildet.

Verzögerte Ausführung

Mit der Funktion 

setTimeout
(einmalige Ausführung) bzw. 
setInterval
(wiederholte Ausführung) kann man Funktionen verzögert (und wiederholt) ausführen lassen. Als erster Parameter wird entweder eine anonyme Funktion übergeben oder der Name einer definierten Funktion und dann noch die Zeit in Millisekunden, wie lange gewartet werden soll.

setInterval

Zufallszahlen

Mit Hilfe der Methode 

Math.random
kann man Zufallszahlen generieren.
